# Visualizers
## 1 Local HTML Report
#### 1.1 插件说明
收集和记录所有 sample的数据保存至本地html文件。

多个脚本的数据能保存至同一个html文件，方便查看所有测试数据。

#### 1.2 使用说明
a. 在 jmeter GUI中右键线程组（Thread）添加-监听器-Local HTML Report；  
b. 在 Local HTML Report GUI中填写 ReportName和 IsAppend；
